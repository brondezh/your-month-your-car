# 🚗 Car Based on Birth Month

This is a simple website that tells you which car matches your personality based on your birth month.

## 🎯 Features
- Select your birth month
- Click the "Spin" button
- Get a random car result
- Each month has multiple possible cars
- Includes personality descriptions

## 🛠️ Built With
- HTML
- CSS
- JavaScript

## 🚀 How to Use
1. Open the website
2. Choose your birth month
3. Click "Spin"
4. See your result
